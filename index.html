Conhecendo as Tags HTML

Esse site vai explicar as 10 tags da linguagem HTML

Lista das Tags:

Abbr: Usada para abreviações e acrônimos

Adress: Essa tag é usada para endereços e dados

Bold: Usada para formatar em negrito

Cite: É uma tag para citar a referência

Code: Serve para encapsular um fragmento de código de computador

Del: Uso para exclusão de texto

Dfn: Tag de definição

Em: Usada para enfatizar determinado conteúdo

I: Tag usada para dar um destaque diferenciado

Kbd: Tag de definição de entrada de teclado

Exemplos:

Abbr: Sr.

Address:

Condominio Mar do Norte, R. Eugênio Fleischer, 8 - Costa e Silva, Joinville - SC, Brasil

Bold: NEGRITO

Cite: Como disse Machado de Assis em Memórias postumas de Brás Cubas: "Não tive filhos, não transmiti a nenhuma criatura o legado da nossa miséria".

Del: Black Friday! 50% off - R$100,00 R$50,00!

Dfn: HTML é uma linguagem de marcação de hipertexto utilizada para estruturar páginas web.

Em: O melhor curso do SESI é Desenvolvimento de Sistemas.

I: O melhor professor do SESI é o Icaro.

Kbd: Clique no Enter para confirmar.
